# Changelog

All notable changes to the Code Annotator extension will be documented in this file.

## [0.1.0] - 2026-03-19

### Added

- Annotate selected code or the current line with free-text comments.
- Background highlight and gutter icon decorations for annotated lines.
- Edit and delete annotations via the editor context menu.
- Collect all annotations as structured markdown copied to the clipboard.
- Collection history with browse and re-copy support (up to 50 snapshots).
- Configurable highlight color and gutter icon color via VS Code settings.
- Status bar item showing annotation count (click to collect).
- Line drift tracking -- annotations follow code as lines are added or removed.
- Keybinding: Cmd+Shift+A (macOS) / Ctrl+Shift+A (Windows/Linux).
